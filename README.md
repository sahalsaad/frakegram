# picbuc

### How to start

1. Clone the project:
       
       git clone https://github.com/sahalsaad/picbuc.git
    
2. Install javascript dependencies:

        yarn install
        
3. Prepare the database:

        bin/rails db:create
        bin/rails db:migrate
        
4. Start the application
        
        bin/rails s
        
        
### Gems in action
    1. react_on_rails
    2. devise
    3. devise-bootstrap-views
    4. carrierwave
    5. mini_magick
